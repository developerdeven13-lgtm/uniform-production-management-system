# Tailoring Workflow

1. Admin creates order
2. Order status = Draft
3. Admin confirms order
4. Tailor Master assigns tailor
5. Tailor updates progress
6. Embroidery team receives order
7. QC verifies order
8. Order marked Ready
9. Delivery completed

# Rules

- Embroidery cannot start before tailoring completion
- QC required before Ready
- Cancelled orders become immutable
